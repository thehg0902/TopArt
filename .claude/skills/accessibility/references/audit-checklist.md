# AA Audit Checklist (run per page)
[ ] Tab through entire page: order logical, focus always visible, no traps
[ ] Zoom 200%: no loss of content/function, no horizontal scroll
[ ] Headings outline reads as a sensible document (h1>h2>h3)
[ ] Landmarks: exactly one main; nav labeled; footer present
[ ] Every control reachable + operable by keyboard (accordion, nav toggle,
    lightbox close on Esc)
[ ] Text over imagery: scrim present, worst-case frame still >= 4.5:1
[ ] Forms: labels, error text, aria-live, autocomplete attrs (tel, email)
[ ] tap targets >= 44px; link text meaningful out of context
[ ] Reduced-motion: verify with emulation - no entrance motion, video still
[ ] Page has lang attr, unique title, and works with JS disabled
