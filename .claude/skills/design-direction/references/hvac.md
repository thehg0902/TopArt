# Niche Playbook: HVAC / Trades
Converts on: trust + speed of contact. Visitor is often in a breakdown.
Direction defaults: clean blues/neutrals + one warm accent; sturdy sans;
straightforward layout, zero cleverness. Hero: static or subtle - big
headline stating service + area + phone. Must-have: phone in header
(tap-to-call), emergency/24-7 badge if true, service area list, licenses/
certifications, reviews, simple quote form (short: name/phone/issue).
Avoid: long forms, animation-heavy pages, ambiguous CTAs.
Must-have sections: hero (phone visible), services, about (licenses/service-area), testimonials, contact (short quote form), cta
