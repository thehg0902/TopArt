# Hero Composition Patterns
Pick per direction + media availability:
1. Full-bleed media + overlay copy: needs strong media; overlay must keep
   AA contrast (gradient scrim token, not opacity guess).
2. Split hero (copy | media): safest for trades/clinics; copy left on LTR.
3. Contained media card on tinted band: premium feel, works with weaker media.
4. Type-only hero: when no good media exists - lean on the distinctive
   typography element; often better than bad stock.
All: H1 + subhead + primary CTA visible without scroll at 360x640;
phone-first niches show tap-to-call beside/above CTA.
