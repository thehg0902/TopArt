# Image Prompt Construction
Structure: [subject + action] + [environment matching the niche] +
[STYLE BLOCK] + [composition note] + [negative/avoid note].
STYLE BLOCK is written ONCE per site from the design direction, e.g.
"warm natural light, shallow depth of field, editorial photography,
muted earthy palette, no text, no watermarks" - and pasted verbatim into
every prompt for consistency.
Composition: leave copy space where layout needs it (hero: subject
off-center toward the media side). Aspect ratios: hero 16:9, cards 4:3
or 1:1 - decide per layout before generating, log in the row.
Ethics: generated people must be obviously ambient/lifestyle, never
presented as the actual staff; interiors labeled as ambiance, not the
actual premises, unless client approves the substitution in writing
(QUESTIONS.md).
