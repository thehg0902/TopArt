# Hero Headline Formulas
Pick by visitor intent:
- Emergency/need-now (HVAC, plumbing): [Service] in [Area]. [Speed/availability proof]. -> phone CTA
- Consideration (dental, gym): [Outcome] without [common anxiety/objection]. -> book CTA
- Atmosphere (cafe, restaurant): evocative fragment naming the experience + place. -> menu/directions CTA
Subheadline carries the specifics the headline omitted (area served,
differentiator, offer). Never make the headline do both jobs.
Test each candidate: could a competitor paste this unchanged? If yes, it's
too generic - add the client's specific proof or place.
