# CTA Copy
Match verb to action: Call (tap-to-call, show number in the button on
mobile), Book (Calendly), Get a quote (form), See the menu, Get directions.
First-person converts in tests for consideration niches ("Book my visit").
One primary CTA style per page; secondary CTA is text link, not a second
button fighting for attention. Repeat primary CTA: hero, after proof
(testimonials), page bottom. Button copy 2-4 words, starts with a verb.
